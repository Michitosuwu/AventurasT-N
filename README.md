# Kitty Adventure

Project made by @Apb8 & @MartaGnarta

## Key Features

 - Run
 - Jump
 - Explore the map as a kitten
 
## Controls

 - W, A, S, D Movement
 - F1 Teleport start level
 - F2 Teleport second level
 - F3 Start from the beginning
 - F9 Visualize the colliders
 - F10 God Mode
 - F11 Enble / Disble FPS
